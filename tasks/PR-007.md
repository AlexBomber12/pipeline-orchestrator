# PR-007: Claude CLI wrapper

Branch: pr-007-claude-cli

## Scope

Create a wrapper module around the `claude` CLI that the daemon will use to trigger coding and fix tasks. Three functions matching the trigger phrases from AGENTS.md.

## Files to create

### src/claude_cli.py

Import logging, subprocess.

Logger: `logging.getLogger(__name__)`

Public functions:

run_claude(prompt: str, cwd: str, timeout: int = 600) -> tuple[int, str, str]
- Build command: ["claude", "--print", "--dangerously-skip-permissions", prompt]
- Run via subprocess.run with capture_output=True, text=True, timeout, cwd
- Log the prompt (first 80 chars) at INFO before running
- Log exit code at INFO after running
- On subprocess.TimeoutExpired: log error, return (-1, "", "Timeout after {timeout}s")
- On FileNotFoundError: log error, return (-1, "", "claude CLI not found")
- Return (returncode, stdout, stderr)

run_planned_pr(repo_path: str) -> tuple[int, str, str]
- Calls run_claude("PLANNED PR", repo_path, timeout=900)

fix_review(repo_path: str) -> tuple[int, str, str]
- Calls run_claude("FIX REVIEW", repo_path, timeout=600)

diagnose_error(repo_path: str, context: str) -> tuple[int, str, str]
- Prompt: "You are the pipeline orchestrator. An infrastructure error occurred. Error context: {context} Respond with exactly one word on the first line: FIX, SKIP, or ESCALATE. If FIX, include a brief action plan on subsequent lines."
- Calls run_claude(prompt, repo_path, timeout=120)

parse_diagnosis(stdout: str) -> str
- Extract first word from stdout, uppercase
- If it starts with "FIX", "SKIP", or "ESCALATE", return that word
- Otherwise return "ESCALATE"

### tests/test_claude_cli.py

- Test run_claude returns correct tuple on success (mock subprocess.run with returncode=0)
- Test run_claude returns (-1, "", "Timeout...") on TimeoutExpired
- Test run_claude returns (-1, "", "claude CLI not found") on FileNotFoundError
- Test run_planned_pr calls run_claude with "PLANNED PR"
- Test fix_review calls run_claude with "FIX REVIEW"
- Test parse_diagnosis("FIX\ndo something") returns "FIX"
- Test parse_diagnosis("SKIP") returns "SKIP"
- Test parse_diagnosis("ESCALATE: too complex") returns "ESCALATE"
- Test parse_diagnosis("I don't know") returns "ESCALATE"
- Test parse_diagnosis("") returns "ESCALATE"

All subprocess.run calls must be mocked via monkeypatch or unittest.mock.patch.
