# PR-001: Project bootstrap

Branch: pr-001-bootstrap

## Scope

Create the project infrastructure: Docker setup, dependency management, CI scripts, and empty source structure. No application logic in this PR.

## Files to create

### Dockerfile
- Base: python:3.12-slim
- Install system deps: git, curl, ca-certificates
- Install gh CLI from official GitHub apt repo
- Install Node.js 22 via nodesource, then npm install -g @anthropic-ai/claude-code
- WORKDIR /app
- Copy and pip install requirements.txt (use --no-cache-dir)
- Copy project files
- Expose 8000
- Default CMD: uvicorn src.web.app:app --host 0.0.0.0 --port 8000

### docker-compose.yml
3 services:
- web: build from Dockerfile, command uvicorn with --reload, port 8000:8000, volumes (config.yml readonly, ./data:/data, .:/app), env REDIS_URL=redis://redis:6379/0, depends on redis
- daemon: build from Dockerfile, command python -m src.daemon.main, volumes (config.yml readonly, ./data:/data), env REDIS_URL + CLAUDE_CONFIG_DIR=/data/auth/claude + GH_CONFIG_DIR=/data/auth/gh, depends on redis
- redis: image redis:7-alpine, volume redis_data:/data
- Define volume redis_data

### config.yml
Default config with empty repositories list, daemon section (poll_interval_sec: 60, review_timeout_min: 60, hung_fallback_codex_review: true, error_handler_use_ai: true), web section (host: 0.0.0.0, port: 8000), auth section (claude_config_dir: /data/auth/claude, gh_config_dir: /data/auth/gh).

### scripts/ci.sh
Executable bash script. Runs: ruff check on src/. If tests/ directory exists, runs pytest -q. Exits with appropriate code.

### scripts/make-review-artifacts.sh
Executable bash script. Creates artifacts/ directory. Runs scripts/ci.sh and saves output to artifacts/ci.log. Generates artifacts/pr.patch via git diff origin/main...HEAD. Generates artifacts/structure.txt via find command (exclude __pycache__, .git, node_modules, artifacts/).
