# PR-038: Canonical repo identity

Branch: pr-038-repo-identity

## Scope

Replace basename-only repo identification with canonical owner__repo format everywhere. Currently `repo_name_from_url("https://github.com/owner-a/api")` returns "api". Two repos with same name but different owners would collide on clone path, Redis keys, upload keys, web routes, and CLI log keys.

## Changes

### src/utils.py

Replace `repo_name_from_url` with `repo_slug_from_url` that returns `owner__repo` format:
```python
def repo_slug_from_url(url: str) -> str:
    cleaned = url.strip().rstrip("/")
    if cleaned.endswith(".git"):
        cleaned = cleaned[:-4]
    parts = cleaned.rsplit("/", 2)
    if len(parts) >= 2:
        owner = parts[-2].rsplit("/", 1)[-1]  # handle github.com/owner/repo
        repo = parts[-1]
        return f"{owner}__{repo}"
    return parts[-1]
```

Keep `repo_name_from_url` as a deprecated alias that calls the new function, or rename all call sites.

### src/daemon/runner.py

Update PipelineRunner.__init__:
```python
self.name = repo_slug_from_url(repo_config.url)  # was repo_name_from_url
```

This automatically propagates to:
- `self.repo_path = f"/data/repos/{self.name}"` (clone path)
- Redis keys: `pipeline:{self.name}`, `cli_log:{self.name}:*`, `upload:{self.name}:*`
- Web routes use `self.name` from Redis state

### src/web/app.py

Update any route matching or state lookup that uses repo name. Routes like `/repo/{name}` and `/partials/repo/{name}/*` should work unchanged since they read the name from Redis state which will now contain the slug.

If there are hardcoded repo name lookups, update them to use the slug format.

### Data migration

Existing Redis keys and clone paths use the old basename format. On first run after this change:
- Clone path `/data/repos/pipeline-orchestrator` won't match `/data/repos/AlexBomber12__pipeline-orchestrator`
- Redis keys won't match

Add a one-time migration in PipelineRunner.__init__ or ensure_repo_cloned:
```python
old_path = f"/data/repos/{old_basename}"
new_path = f"/data/repos/{self.name}"
if Path(old_path).exists() and not Path(new_path).exists():
    shutil.move(old_path, new_path)
    self.log_event(f"Migrated clone path {old_path} -> {new_path}")
```

For Redis: on first publish_state, the old key becomes orphaned. Clean up old key:
```python
old_key = f"pipeline:{old_basename}"
if await self.redis.exists(old_key):
    await self.redis.delete(old_key)
```

- test_repo_slug_from_url_strips_git: handles .git suffix

Update _make_runner and any tests that assert on self.name to expect owner__repo format.
