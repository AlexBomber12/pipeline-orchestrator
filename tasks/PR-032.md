# PR-032: Subprocess helper and error handling hardening

Branch: pr-032-subprocess-helper

## Scope

Extract repeated subprocess.run patterns into a helper. Add OSError handling where missing. Fixes review findings on inconsistent error handling.

## Changes

### src/daemon/runner.py

1. Add module-level helper:
```python
def _git(
    repo_path: str, *args: str, timeout: int = 30
) -> subprocess.CompletedProcess[str]:
    return subprocess.run(
        ["git", *args],
        capture_output=True,
        text=True,
        timeout=timeout,
        check=True,
        cwd=repo_path,
    )
```

2. Replace all bare subprocess.run(["git", ...]) calls in runner.py with _git(). This includes calls in: sync_to_main, _base_branch_ahead_of_origin, _working_tree_dirty, ensure_repo_cloned, _parse_base_queue, preflight, _commit_and_push_dirty, _mark_queue_done (from PR-030), handle_coding, handle_fix, and any other methods.

3. Keep non-git subprocess calls (scripts/ci.sh, claude CLI) as-is since they have different argument patterns.

4. Add OSError to except clauses in preflight() and sync_to_main() where it is currently missing. The _git helper raises OSError for missing git binary or missing cwd. Without catching it, the exception escapes to daemon.main's generic handler and the runner state becomes stale.

5. In preflight(), update the except to:
```python
except (subprocess.CalledProcessError, subprocess.TimeoutExpired, OSError) as exc:
```

6. In sync_to_main(), wrap the whole method body in:
```python
try:
    ...
except OSError as exc:
    raise RuntimeError(f"sync_to_main OS error: {exc}") from exc
```

### src/queue_parser.py

Fix the silent TODO fallback for unknown status values (review finding). Change:
```python
status = TaskStatus(raw_status) if raw_status in TaskStatus.__members__.values() else TaskStatus.TODO
```
to:
```python
if raw_status in {s.value for s in TaskStatus}:
    status = TaskStatus(raw_status)
else:
    logger.warning("Unknown status %r for %s, treating as TODO", raw_status, pr_id)
    status = TaskStatus.TODO
```

Add `import logging` and `logger = logging.getLogger(__name__)` at top of file.

### tests/test_runner.py

- Update existing tests that mock subprocess.run for git commands to work with the _git helper. The mock target stays the same (subprocess.run) since _git calls subprocess.run internally.
- test_preflight_handles_oserror: mock subprocess.run to raise OSError, verify ERROR state
- test_sync_to_main_handles_oserror: mock subprocess.run to raise OSError, verify RuntimeError raised

### tests/test_queue_parser.py

- test_unknown_status_defaults_to_todo_with_warning: parse queue with "Status: INVALID", verify task.status == TODO and warning logged
