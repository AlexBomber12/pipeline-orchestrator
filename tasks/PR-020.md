## Scope

Small fixes identified during sprint review and first deployment.

## Changes

### 1. Update .gitignore

Add to .gitignore:
```
.claude/
*:Zone.Identifier
```

Delete any existing `.claude/` directory and `*:Zone.Identifier` files from the repo (git rm).

### 2. Docker compose port via .env

Create `.env.example`:
```
WEB_PORT=8800
```

Modify docker-compose.yml: change ports from `"8000:8000"` to `"${WEB_PORT:-8800}:8000"`. Default is 8800 (matching the deployed server).

Add `.env` to .gitignore if not already there.

### 3. Pipe stdin from /dev/null in claude CLI

File: src/claude_cli.py

Modify run_claude: add `stdin=subprocess.DEVNULL` to the subprocess.run call. This prevents Claude CLI from hanging if it tries to read user input. Any prompt for confirmation will get EOF and Claude will be forced to continue or exit.

### 4. Update tests

Modify tests/test_claude_cli.py: verify that subprocess.run is called with stdin=subprocess.DEVNULL in the mocked calls.
