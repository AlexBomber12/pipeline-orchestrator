# PR-171: Spinners on upload and settings forms

Branch: pr-171-spinners-upload-settings
- Type: ux
- Complexity: low
- Depends on: none
- Priority: 2
- Coder: any

## Problem

When user clicks Upload or Save Settings button, there is no visual feedback that the action is in progress. HTMX request is in flight, but the button looks the same. Users either click again (causing duplicate submissions) or stare at unchanged UI wondering if the click registered.

## Scope

Add HTMX hx-indicator pattern to the upload form and settings form. Show a small spinner inside or next to the button while request is in flight. Use a single shared spinner partial so behavior is consistent across forms.

This PR addresses upload form + settings form only. Other actions (controls, coder selector) covered in PR-172.

## Files to create

`src/web/templates/components/_spinner.html`: small reusable spinner partial. Sketch:

```html
<span class="htmx-indicator inline-flex items-center" aria-hidden="true">
    <svg class="animate-spin h-4 w-4 text-current" xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24">
        <circle class="opacity-25" cx="12" cy="12" r="10" stroke="currentColor" stroke-width="4"></circle>
        <path class="opacity-75" fill="currentColor" d="M4 12a8 8 0 018-8v4a4 4 0 00-4 4H4z"></path>
    </svg>
</span>
```

The `.htmx-indicator` class is HTMX-built-in: visible only when the associated request is in flight.

## Files to touch

`src/web/templates/base.html`: ensure HTMX htmx-indicator default CSS is present (hidden when not in flight). HTMX 1.x ships this by default; verify.

`src/web/templates/index.html` (or wherever upload form lives): in the upload form's submit button, add `hx-indicator="#upload-spinner"` and place `{% include "components/_spinner.html" %}` with `id="upload-spinner"` next to the button text.

`src/web/templates/settings.html` (or wherever settings form lives): same treatment for Save Settings button.

`tests/test_templates.py` (or `tests/test_dashboard.py`): assert the upload and settings buttons render with `hx-indicator` attribute and the spinner partial is included.
