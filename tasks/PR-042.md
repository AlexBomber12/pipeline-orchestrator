# PR-042: Async Claude CLI and per-repo scheduling

Branch: pr-042-async-cli

## Scope

Replace blocking subprocess.run in Claude CLI calls with asyncio.create_subprocess_exec. Currently one 30-minute FIX REVIEW blocks the entire daemon: no state updates, no dashboard refresh, no other repos processed. Also implement per-repo poll intervals so repos can have different check frequencies.

## Changes

### src/claude_cli.py

Add async version of run_claude:
```python
async def run_claude_async(
    prompt: str,
    cwd: str,
    timeout: int = 600,
    model: str | None = None,
) -> tuple[int, str, str]:
    cmd = ["claude", "--print", "--dangerously-skip-permissions"]
    if model:
        cmd.extend(["--model", model])
    cmd.append(prompt)
    logger.info("running claude CLI with prompt: %s", prompt[:80])
    try:
        proc = await asyncio.create_subprocess_exec(
            *cmd,
            stdout=asyncio.subprocess.PIPE,
            stderr=asyncio.subprocess.PIPE,
            cwd=cwd,
            stdin=asyncio.subprocess.DEVNULL,
        )
        stdout_bytes, stderr_bytes = await asyncio.wait_for(
            proc.communicate(), timeout=timeout
        )
        stdout = stdout_bytes.decode("utf-8", errors="replace")
        stderr = stderr_bytes.decode("utf-8", errors="replace")
        code = proc.returncode or 0
    except asyncio.TimeoutError:
        proc.kill()
        await proc.wait()
        logger.error("claude CLI timed out after %ss", timeout)
        return (-1, "", f"Timeout after {timeout}s")
    except FileNotFoundError as exc:
        missing = getattr(exc, "filename", "")
        if missing and missing != cmd[0]:
            return (-1, "", f"cwd not found: {missing}")
        return (-1, "", "claude CLI not found")
    logger.info("claude CLI exited with code %s", code)
    return (code, stdout, stderr)
```

Add async wrappers:
```python
async def run_planned_pr_async(repo_path: str, model: str | None = None) -> tuple[int, str, str]:
    return await run_claude_async("PLANNED PR", repo_path, timeout=900, model=model)

async def fix_review_async(repo_path: str, model: str | None = None) -> tuple[int, str, str]:
    return await run_claude_async("FIX REVIEW", repo_path, timeout=1800, model=model)

async def diagnose_error_async(repo_path: str, context: str, model: str | None = None) -> tuple[int, str, str]:
    prompt = (
        "You are the pipeline orchestrator. An infrastructure error occurred. "
        f"Error context: {context} "
        "Respond with exactly one word on the first line: FIX, SKIP, or ESCALATE. "
        "If FIX, include a brief action plan on subsequent lines."
    )
    return await run_claude_async(prompt, repo_path, timeout=120, model=model)
```

Keep synchronous versions for backward compatibility (merge conflict resolution in handle_merge uses sync call in a subprocess context).

### src/daemon/runner.py

1. Replace all `claude_cli.run_planned_pr(...)` with `await claude_cli.run_planned_pr_async(...)`
2. Replace all `claude_cli.fix_review(...)` with `await claude_cli.fix_review_async(...)`
3. Replace all `claude_cli.diagnose_error(...)` with `await claude_cli.diagnose_error_async(...)`

Exception: the merge conflict resolution in handle_merge can stay sync (short timeout, runs rarely).

4. In handle_coding and handle_fix, add periodic state publishing while Claude runs. Use asyncio.create_task to publish state every 30 seconds:
```python
async def _publish_while_waiting(self, label: str):
    while True:
        await asyncio.sleep(30)
        self.log_event(f"{label}...")
        await self.publish_state()
```
Start before Claude CLI call, cancel after:
```python
heartbeat = asyncio.create_task(self._publish_while_waiting("Claude working"))
try:
    code, stdout, stderr = await claude_cli.run_planned_pr_async(...)
finally:
    heartbeat.cancel()
```

### src/daemon/main.py

Add per-repo poll interval support. Instead of sleeping daemon.poll_interval_sec for all repos, track last_run per runner and skip repos whose interval hasn't elapsed:

```python
last_run: dict[str, float] = {}

for key, runner in list(runners.items()):
    interval = runner.repo_config.poll_interval_sec
    now = time.monotonic()
    if now - last_run.get(key, 0) < interval:
        continue
    last_run[key] = now
    # ... run_cycle
```

Add `poll_interval_sec` back to RepoConfig (removed in PR-037) now that it's actually used.
