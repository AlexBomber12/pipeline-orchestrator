# PR-016: Settings page - daemon and auth status

Branch: pr-016-settings-daemon

## Scope

Add daemon settings section and auth status display to the Settings page.

## Files to modify

### src/web/app.py

Add routes:

PUT /settings/daemon
- Body: poll_interval_sec, review_timeout_min, hung_fallback_codex_review, error_handler_use_ai (all optional)
- Call config.update_daemon_config(**updates)
- Return updated daemon settings partial

GET /partials/settings/daemon -> daemon settings component

GET /api/auth-status -> JSON
- Check claude CLI: run `claude --version` (subprocess, timeout 5s). Success = authorized, failure = not found/error.
- Check gh CLI: run `gh auth status` (subprocess, timeout 5s). Parse output for "Logged in" = authorized, otherwise expired/not configured.
- Return {"claude": {"status": "ok|error", "detail": "..."}, "gh": {"status": "ok|error", "detail": "..."}}

GET /partials/settings/auth-status -> auth status component (for HTMX polling)

### src/web/templates/settings.html

Add sections after Repositories:

Section "Daemon Settings":
- Poll interval (number input, seconds) with hx-put on blur/enter
- Default review timeout (number input, minutes)
- HUNG fallback: post @codex review (toggle)
- Error handler: use AI diagnosis (toggle)
- Each toggle/input sends PUT /settings/daemon via HTMX on change

Section "Auth Status":
- Claude CLI: status dot (green/red) + version or error message
- GitHub CLI: status dot (green/red) + "Logged in as {user}" or error message
- Auto-refresh via hx-get="/partials/settings/auth-status" hx-trigger="every 30s"
- Note below: "Run `docker compose run --rm daemon bash` to re-authenticate"

### src/web/templates/components/settings_daemon.html
- Form layout matching repo settings style
- Toggles for booleans, number inputs for integers
- Each field sends HTMX PUT on change

### src/web/templates/components/settings_auth.html
- Two cards side by side (Claude CLI / GitHub CLI)
- Status dot: green for ok, red for error
- Detail text: version/user or error message
- Monospace for version strings
