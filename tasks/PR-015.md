# PR-015: Settings page - repositories

Branch: pr-015-settings-repos

## Scope

Create the Settings page with repository management: list, add, edit, delete. Uses HTMX for inline editing without full page reloads.

## Files to modify

### src/web/app.py

Add routes:

GET /settings -> settings.html
- Load config, render settings page with repo list and daemon settings

POST /settings/repos
- Body: url (required), branch (optional, default "main"), auto_merge (optional, default true)
- Call config.add_repository(url, ...)
- Return updated repo list partial (HTMX swap)
- On error (duplicate URL): return error message partial with 422 status

DELETE /settings/repos/{name}
- name = repo name (from URL)
- Find repo URL by name, call config.remove_repository(url)
- Return updated repo list partial
- On error: return error message partial with 404 status

PUT /settings/repos/{name}
- Body: branch, auto_merge, review_timeout_min, poll_interval_sec (all optional)
- Find repo URL by name, call config.update_repository(url, **updates)
- Return updated single repo row partial
- On error: return error message partial

GET /partials/settings/repo-list -> settings repo list component (for HTMX refresh)

## Design notes

- Table with subtle borders (border-white/5)
