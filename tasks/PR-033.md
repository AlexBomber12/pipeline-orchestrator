# PR-033: Dirty tree recovery and dedup codex review

Branch: pr-033-dirty-recovery

## Scope

Two daemon resilience improvements:

1. Auto-recovery from dirty working tree. If daemon is stuck in "preflight: dirty working tree" for N consecutive cycles, automatically reset the tree and return to IDLE. Currently this state requires manual intervention.

2. Deduplicate @codex review comments. Claude posts @codex review as part of AGENTS.md runbook. Daemon also posts via _post_codex_review after handle_coding and handle_fix. This creates duplicate comments.

## Changes

### src/daemon/runner.py

1. Add a counter to PipelineRunner.__init__:
```python
self._consecutive_dirty_cycles = 0
```

2. In preflight(), when dirty tree is detected:
```python
self._consecutive_dirty_cycles += 1
if self._consecutive_dirty_cycles >= 3:
    self.log_event(
        f"Dirty tree persisted {self._consecutive_dirty_cycles} cycles, "
        "auto-resetting to recover"
    )
    try:
        branch = self.repo_config.branch
        subprocess.run(
            ["git", "checkout", branch],
            capture_output=True, text=True, timeout=30,
            check=False, cwd=self.repo_path,
        )
        subprocess.run(
            ["git", "reset", "--hard", f"origin/{branch}"],
            capture_output=True, text=True, timeout=30,
            check=True, cwd=self.repo_path,
        )
        subprocess.run(
            ["git", "clean", "-fd"],
            capture_output=True, text=True, timeout=30,
            check=True, cwd=self.repo_path,
        )
        self._consecutive_dirty_cycles = 0
        self.state.state = PipelineState.IDLE
        self.state.error_message = None
        self.log_event("Auto-recovered from dirty tree -> IDLE")
        return True
    except Exception as exc:
        self.log_event(f"Auto-recovery failed: {exc}")
```
When preflight succeeds, reset the counter:
```python
self._consecutive_dirty_cycles = 0
```

3. In _post_codex_review, before posting, check if a recent @codex review comment from the PR author already exists:
```python
def _post_codex_review(self, pr_number: int) -> bool:
    try:
        if github_client.has_recent_codex_review_request(
            self.owner_repo, pr_number,
            pr_author=self._get_pr_author(),
            within_minutes=5,
        ):
            self.log_event(
                f"Skipping duplicate @codex review on PR #{pr_number}"
            )
            return True
    except Exception:
        pass

    try:
        github_client.post_comment(...)
```

Add helper _get_pr_author that returns the GitHub login from the auth context or "AlexBomber12" as fallback.

### src/github_client.py

Add function:
```python
def has_recent_codex_review_request(
    repo: str,
    pr_number: int,
    pr_author: str,
    within_minutes: int = 5,
) -> bool:
    comments = _gh_api_paginated(
        f"repos/{repo}/issues/{pr_number}/comments"
    ) or []
    now = datetime.now(timezone.utc)
    for c in reversed(comments):
        author = (c.get("user") or {}).get("login", "")
        if author != pr_author:
            continue
        body = (c.get("body") or "").lower()
        if "@codex review" not in body:
            continue
        created = _parse_iso(c.get("created_at"))
        if created and (now - created).total_seconds() < within_minutes * 60:
            return True
    return False
```

Add `from datetime import timezone` if not already imported.
