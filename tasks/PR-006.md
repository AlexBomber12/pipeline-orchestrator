# PR-006: Dashboard repo detail and HTMX polling

Branch: pr-006-dashboard-detail

## Scope

Create the repository detail page showing current task, current PR status (CI + review), event log, and HTMX auto-refresh every 5 seconds.

## Files to create

### src/web/templates/repo.html
- Extends base.html
- Title block: "{repo.name} - Pipeline Orchestrator"
- Back link to / (arrow + "All repositories")
- Content div with hx-get="/partials/repo/{name}" hx-trigger="every 5s" hx-swap="innerHTML"
- Include components/repo_detail.html

### src/web/templates/components/repo_detail.html

Layout:
- Header: repo name (font-mono, text-2xl) + state badge (colored) + last updated timestamp
- Error banner (if error_message): red background, error text in monospace
- Two-column grid (stack on mobile):
  - Left card "Current Task": pr_id, title, task_file path
  - Right card "Current PR": PR number (link to GitHub URL), branch name, CI status dot (green/red/yellow), Review status emoji (thumbs up/eyes/pencil/gray dot), push count
- Event log card: scrollable list (max-h-96), each entry shows time (HH:MM:SS), state badge (colored), event text. Entries in reverse chronological order. Empty state if no events.

### Routes to add in src/web/app.py

- GET /repo/{name} -> repo.html
- GET /partials/repo/{name} -> repo_detail.html component (for HTMX swap)

Both routes: read RepoState from Redis by key "pipeline:{name}". If not found, return default RepoState.

## Design notes
- Same dark theme as index page
- PR number is a clickable link opening GitHub in new tab
- CI status: green dot for SUCCESS, red dot for FAILURE, yellow pulsing dot for PENDING
- Review status: thumbs up emoji for APPROVED, eyes emoji for EYES, pencil emoji for CHANGES_REQUESTED, gray dot for PENDING
- Event log uses monospace font, compact rows with subtle bottom border
- State badges use same color scheme as repo list cards

## Success criteria
- scripts/ci.sh exits 0
- GET /repo/test-repo returns HTML page
- GET /partials/repo/test-repo returns HTML fragment
- HTMX auto-refreshes detail every 5 seconds
- Clicking back arrow returns to index
- PR number links to correct GitHub URL
