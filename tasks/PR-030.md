# PR-030: QUEUE.md auto-update and squash merge

Branch: pr-030-queue-done-squash

## Scope

Two related merge-time improvements:

1. After successful merge, daemon updates QUEUE.md to mark the task DONE. Claude sometimes skips this AGENTS.md step, causing daemon to re-pick merged tasks.

2. Switch from --merge to --squash so fix iterations don't pollute main history.

## Changes

### src/github_client.py

1. Change merge_pr to use --squash:
```python
def merge_pr(repo: str, pr_number: int) -> None:
    run_gh(["pr", "merge", str(pr_number), "--squash", "--delete-branch"], repo=repo)
```

### src/daemon/runner.py

In handle_merge, after successful github_client.merge_pr and before setting state to IDLE:

```python
try:
    self._mark_queue_done()
except Exception as exc:
    self.log_event(f"Warning: QUEUE.md update failed: {exc}")
```

Add method _mark_queue_done:
```python
def _mark_queue_done(self) -> None:
    if self.state.current_task is None:
        return
    pr_id = self.state.current_task.pr_id
    branch = self.repo_config.branch

    subprocess.run(
        ["git", "fetch", "origin", branch],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )
    subprocess.run(
        ["git", "checkout", branch],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )
    subprocess.run(
        ["git", "reset", "--hard", f"origin/{branch}"],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )

    queue_path = Path(self.repo_path) / "tasks" / "QUEUE.md"
    if not queue_path.exists():
        return
    content = queue_path.read_text()

    import re
    pattern = rf"(## {re.escape(pr_id)}:.*?\n- Status: )(?:TODO|DOING)"
    updated = re.sub(pattern, r"\1DONE", content, count=1)
    if updated == content:
        return

    queue_path.write_text(updated)
    subprocess.run(
        ["git", "add", "tasks/QUEUE.md"],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )
    subprocess.run(
        ["git", "commit", "-m", f"{pr_id}: mark DONE"],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )
    subprocess.run(
        ["git", "push", "origin", branch],
        capture_output=True, text=True, timeout=30,
        check=True, cwd=self.repo_path,
    )
    self.log_event(f"Marked {pr_id} DONE in QUEUE.md")
```

Important: _mark_queue_done failures must NOT set ERROR state. The PR is already merged. This is best-effort cleanup. Wrap the entire call in try/except in handle_merge and only log a warning.
